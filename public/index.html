Quick Start

Three calls. Full identity.

Check score, register genesis position, verify certificate. No SDK required.

bash GET /api/score 
# KYA score lookup
curl "https://verun-genesis.vercel.app/api/score?address=YOUR_ADDRESS"

# Response
{
  "score":     820,
  "tier":      "TRUSTED",
  "permitted": true,
  "kickback":  "10%",
  "genesis": {
    "claim_id": "vga-0042"
  }
}
bash POST /api/register 
# Claim genesis position
curl -X POST "https://verun-genesis.vercel.app/api/register" \
  -H "Content-Type: application/json" \
  -d '{"address":"GXXX...","addressType":"stellar","inviteCode":"VERUN-FOUNDING-001"}'

# Response
{
  "claim_id":     "vga-0001",
  "score":        350,
  "stellar_anchor": {
    "tx_hash": "a3f9c2d1e5..."
  }
}

MCP Tools

Four tools. Native agent integration.

Add Verun to any MCP-compatible agent runtime. No API key management required.

json mcp.json config 
{
  "servers": {
    "verun-kya": {
      "url": "https://verun-genesis.vercel.app/api/mcp",
      "type": "http"
    }
  }
}

Verifies a KYA certificate by claim ID. Returns validity, score, and Stellar anchor proof.

Returns current Genesis registration count, remaining spots, and network status.
