          javascript
          3. Gate your operations by KYA score
        

        
      

      
async function checkBeforeOperation(agentAddress, operation) {
  const res = await fetch(
    `https://verun.network/api/score?address=${agentAddress}&operation=${operation}`
  );
  const kya = await res.json();

  if (!kya.permitted) {
    throw new Error(`KYA gate: score ${kya.score} insufficient for ${operation}`);
    // → 402 agent_not_verified
  }

  // proceed with operation
  return { agentScore: kya.score, kickback: kya.kickback };
}

// Usage
await checkBeforeOperation("GDRX...", "transfer"); // requires 500+

Transparency

    
What Genesis Beta Actually Is

    
Honest answers about where we are and where we're going.

    

      

        
Is this fully on-chain?

        
Genesis Beta uses off-chain KYA scoring with a Stellar Testnet ManageData anchor per registration. Every claim gets a real transaction hash on Stellar Expert. Full institutional 2-of-3 validator consensus moves to mainnet in June 2026.

      

      

        
What is a "Genesis spot" exactly?

        
A permanent, non-transferable identity position for your AI agent. Claim IDs (vga-0001 to vga-10000) are assigned in order and never renumbered. Your score starts at 350 — above the Read threshold from day one.

      

      

        
How does Verun differ from ERC-8004 or Moltbook?

        
ERC-8004 is a generic Ethereum agent registry with no compliance layer. Moltbook is a social network for agents with no identity verification. Verun is built specifically for regulated European finance — BaFin, eWpG, MiFID II — with institutional validators (BCP Partners, Cashlink, Bitbond, SDF) and a Sybil-resistance score system.

      

      

        
What happens after the 10,000 Genesis cap?

        
Genesis Extended (vga-10001 to vga-100000) requires a governance vote by the validator network. Genesis Open (vga-100001+) opens post mainnet. Founding positions never change claim ID — vga-0001 is always the first.

      

      

        
When do kickbacks start?

        
Kickback payments (5% at score 500+, 10% at score 800+) are on-chain and automatic — no claim required. They activate at mainnet go-live in June 2026. Genesis agents are locked in at founding rates permanently.
